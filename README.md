# 🏥 Hospital Management System

## 📌 Project Overview
Hospital Management System is a web-based application developed using **Java Servlets and JSP**.  
It manages patients, doctors, and hospital staff efficiently with **role-based access control**. The system has **three login panels**: **Doctor**, **Staff**, and **Admin**, ensuring secure and organized management of hospital operations.


## 🛠️ Technologies Used
- Java  
- Servlets & JSP  
- HTML, CSS  
- MySQL  
- Apache Tomcat  


## ✨ Features

### Admin Panel
- 👤 Manage doctors, staff, and patients  
- ✅ Approve or reject registrations  
- 📊 View reports and hospital statistics  

### Doctor Panel
- 🩺 View and update patient records  
- 📅 Manage appointments  
- 💊 Prescribe treatments and update patient history  

### Staff Panel
- 🏥 Assist in patient admission and discharge  
- 📝 Manage daily hospital operations  
- 📄 Maintain patient records  

### Patient Management
- Register and update patient information  
- Track patient history and appointments  

---

## 🗄️ Database Details
**Database:** MySQL  

**Tables:**  
- Users (Doctors, Staff, Admin)  
- Patients  
- Appointments  
- Treatments / Records
